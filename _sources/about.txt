DHCPAP
=======

This Web site contains information about implementations and documentation
related to the DHCP Anonymity Profiles :rfc:`7844`.

``DHCPAP`` stands for DHCP Anonymity Profiles

``DHCPcAP`` stands for DHCP client Anonymity Profiles

``dhcpcanon`` is the name of the Python DHCP client implementation
