    Glosary¶

    • DHCP: Dynamic Host Configuration Protocol.
    • Anonymity Profiles: DHCP client profiles designed to minimize disclosure of identifying information.
    • DHCPAP: stands for DHCP Anonymity Profiles. It’s the name of an Github organization to group work related to the DHCP Anonymity Profiles.
    • DHCPcAP: stands for DHCP client Anonymity Profiles
    • `dhcpcanon <https://dhcpcanon.readthedocs.io/>`__: name created from DHCP and ANONymity, is the name of the Python DHCP client implementation of the Anonymity Profiles
    • dhcpcfp: standos for DHCP client Fingerprint, a DHCP scanner
    • dhcpcfpdb, stands for DHCP client fingerprint database. It’s Web application not in use.
